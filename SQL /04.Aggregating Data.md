# üü¶ **Aggregating Data**

Aggregation means:
üëâ ‚ÄúCombine multiple rows and calculate something from them (like sum, average, count).‚Äù

---

## **1) COUNT()**

**Meaning:**
‚ÄúCount how many rows are there.‚Äù

**Explanation:**

* COUNT(*) counts all rows.
* COUNT(column) counts only non-NULL values of that column.

**Example:**

```sql
SELECT COUNT(*) FROM users;
```

This means:
‚û°Ô∏è Tell me the total number of users.

---

## **2) SUM()**

**Meaning:**
‚ÄúAdd all the values together.‚Äù

**Example:**

```sql
SELECT SUM(amount) FROM orders;
```

This means:
‚û°Ô∏è Add up the amount of all orders and give me the total sales.

---

## **3) AVG()**

**Meaning:**
‚ÄúCalculate the average value.‚Äù

**Example:**

```sql
SELECT AVG(age) FROM users;
```

This means:
‚û°Ô∏è Add all ages and divide by how many users exist.

---

## **4) MIN() and MAX()**

**Meaning:**
‚ÄúFind the smallest or largest value.‚Äù

**Example:**

```sql
SELECT MIN(price), MAX(price) FROM products;
```

This means:
‚û°Ô∏è Show me the cheapest product and the most expensive product.

---

## **5) GROUP BY**

**Meaning:**
‚ÄúGroup rows that have the same value, then apply aggregation.‚Äù

**Explanation:**

* GROUP BY is used with COUNT, SUM, AVG, etc.
* It collects rows with common values.
* Each group gives **one output row**.

**Example:**

```sql
SELECT city, COUNT(*) 
FROM users 
GROUP BY city;
```

This means:
‚û°Ô∏è Group users city-wise and show how many users live in each city.

---

## **6) HAVING**

**Meaning:**
‚ÄúFilter groups after grouping.‚Äù

**Explanation:**

* WHERE ‚Üí filters rows **before** grouping
* HAVING ‚Üí filters groups **after** grouping

**Example:**

```sql
SELECT city, COUNT(*) 
FROM users 
GROUP BY city 
HAVING COUNT(*) > 5;
```

This means:
‚û°Ô∏è Show only those cities where more than 5 users live.

---

# üî• Quick Difference: WHERE vs HAVING

| Clause | Filters         | Used With       |
| ------ | --------------- | --------------- |
| WHERE  | Before grouping | Individual rows |
| HAVING | After grouping  | Groups          |
