**Explanation:**

‚û°Ô∏è Show only those orders where the amount is greater than 500.

## **8) EXISTS**

**Meaning:**
‚ÄúReturn this row only if a related row exists in another table.‚Äù

**Explanation:**

* EXISTS checks if the subquery returns *at least one row*.
* If yes ‚Üí TRUE
* If no ‚Üí FALSE

**Example:**

```sql

FROM users u

    SELECT 1 FROM orders o WHERE o.user_id = u.id

```

This means:
‚û°Ô∏è Show users only if they have at least one order.
